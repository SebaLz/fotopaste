Usa Ctrl+V (Cmd+V en Mac) para pegar una imagen

💡 Tips para capturar pantalla

Windows + Shift + S: Captura rápida que se guarda en el portapapeles
Snipping Tool: Herramienta nativa de Windows para capturas
PrtScn: Captura toda la pantalla al portapapeles
Alt + PrtScn: Captura solo la ventana activa

¡Después de capturar, simplemente usa Ctrl+V aquí!

¿Cómo usar?

  1. Pega una imagen usando Ctrl+V o selecciona un archivo
  2. La imagen se mostrará expandida automáticamente
  3. Haz clic en la imagen para expandir/contraer
  4. Clic derecho para más opciones (abrir, copiar URL, descargar)
  5. Para usar Google Lens: copia la URL y pégalo manualmente en Google Lens
